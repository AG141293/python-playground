# 🐍 PyNest — Python Playground

A lightweight, browser-based Python coding environment. No installation needed — just open and code!

## ✨ Features
- **Script mode** — full editor with syntax highlighting
- **Notebook mode** — Colab-style cells (run independently)
- **Upload .py files** — load your existing scripts
- **Download** — save your work anytime
- **Multiple files** — create and switch between files
- **API Key storage** — saved securely in your browser
- **Package installer** — install numpy, pandas and more
- **Variables panel** — inspect values after running
- **Keyboard shortcuts** — Ctrl+Enter to run, Ctrl+S to save

## 🚀 Live Demo
[Click here to open the playground](https://AG141293.github.io/python-playground/)

## 🛠️ Built With
- HTML, CSS, JavaScript
- [Pyodide](https://pyodide.org) — Python in the browser (WebAssembly)
- [CodeMirror](https://codemirror.net) — code editor

## 📌 Notes
- First load takes ~15 seconds (downloading Python runtime)
- Works with most popular packages: numpy, pandas, matplotlib, requests
- API key is stored only in your browser, never sent anywhere
